# rfsd-web-page-eleventy

The web page for Rapid Fullstack Development.

## Setup

Clone the repo then install dependencies:

```bash
pnpm install
```

## Run in development

```bash
npm start
```

## Run in production

```bash
npm run build
```

## Rebuild css

```bash
npm run build-css
```
